1. Aceptación de los Términos

Al habilitar y usar "Actividades Eminus", aceptas cumplir con estos términos de uso y nuestra política de privacidad.

2. Requisitos para Usar la Skill

Requisitos obligatorios:
  • Cuenta activa de Eminus UV: Debes ser estudiante o personal de la Universidad Veracruzana
  • Credenciales válidas: Usuario y contraseña funcionales del sistema Eminus
  • Dispositivo compatible: Dispositivo Amazon Echo o app de Alexa
  • Conexión a internet: Para acceder a los datos de Eminus
  • Mayoría de edad: 18 años o más (o con permiso de tutor legal)

3. Uso Permitido

Puedes usar esta skill para:

  • Consultar tus tareas y actividades pendientes
  • Obtener información de tus cursos favoritos
  • Revisar detalles específicos de cursos (profesores, módulos, exámenes)
  • Recibir notificaciones sobre fechas de entrega
  • Acceder a tu información académica personal

NO está permitido:

  • Extraer datos masivamente para fines comerciales
  • Reproducir o distribuir la información académica de terceros
  • Usar la skill para fines ilegales o no autorizados

5. Responsabilidades del Usuario

  • Mantener seguras tus credenciales de Eminus
