Política de Privacidad

Actividades Eminus - Alexa Skill

Última actualización: 29 de noviembre de 2025

1. Información que Recopilamos

Importante: Esta skill requiere vinculación de cuenta (account linking) para funcionar.
  • Credenciales de Eminus: Nombre de usuario y contraseña del sistema Eminus UV
  • Tokens de acceso: Tokens JWT generados por Eminus para autenticación
  • Información académica:
    • Nombres de cursos y materias
    • Lista de cursos favoritos
    • Títulos y descripciones de actividades/tareas
    • Fechas de entrega de actividades
    • Nombres de profesores y facilitadores
    • Calificaciones de exámenes
    • Estado de actividades (pendientes/completadas)
  • Información de sesión: Datos temporales de sesión durante el uso de la skill

2. Cómo Usamos tu Información

  • Autenticación: Las credenciales se usan únicamente para obtener tokens de acceso de Eminus
  • Funcionalidad principal: La información académica se usa para:
    • Proporcionar listados de tareas pendientes por voz
    • Informar sobre cursos favoritos
    • Detallar información específica de cursos (profesores, módulos, exámenes)
    • Notificar sobre fechas de entrega próximas
  • Mejora del servicio: Registros de uso anónimos para mejorar la experiencia

3. Almacenamiento y Retención de Datos

  • Credenciales: Se almacenan temporalmente en sesiones del servidor (máximo 24 horas, o 180 días si selecciona "recordarme")
  • Tokens de acceso: Se gestionan a través del sistema seguro de Alexa
  • Información académica: No se almacena permanentemente; se obtiene en tiempo real de Eminus
  • Registros: Se conservan durante 30 días para diagnóstico técnico

4. Compartición de Información

No compartimos tu información personal con terceros, excepto:

  • Eminus UV: Para autenticación y obtención de datos académicos
  • Amazon: Para el funcionamiento del sistema de Alexa (tokens y gestión de cuentas)

5. Seguridad

  • Conexión HTTPS en todas las comunicaciones
  • Cifrado de credenciales durante transmisión
  • Tokens de acceso con tiempo de expiración limitado
  • Sesiones del servidor con configuración segura

6. Derechos del Usuario

  • Acceso: Puedes revisar qué información se está utilizando en cualquier momento
  • Rectificación: Actualiza tus credenciales si cambian
  • Supresión: Puedes desvincular tu cuenta en cualquier momento desde la app de Alexa
  • Portabilidad: La información pertenece a tu cuenta de Eminus
